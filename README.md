# Integracion-de-modulos-GA8-220501096-AA1-EV01
Sistema de Gestión Logística – SENA
Proyecto desarrollado como evidencia de integración de módulos de software dentro del programa Análisis y Desarrollo de Software – SENA

Sistema web orientado a la gestión logística, que permite administrar:

Bodegas - Productos - Clientes - Coordinadores de despacho - Órdenes de despacho

El sistema integra todos los módulos mediante claves foráneas en una base de datos relacional MySQL, garantizando coherencia y centralización de la información.

Tecnologías
PHP
MySQL
XAMPP
HTML5
CSS3
Bootstrap
JavaScript
Git & GitHub
Funcionalidades
CRUD de Bodegas
CRUD de Productos
CRUD de Clientes
CRUD de Coordinadores
Gestión de Órdenes de Despacho
Integración de módulos mediante claves foráneas
Registro y consulta de Órdenes de Despacho
Consultas con JOIN para integración de información
Chatbot integrado
